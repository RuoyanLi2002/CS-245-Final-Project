# CS-245-Final-Project

# Installation and Usage Guide

## 1. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 2. Run the LLM Vote Agent
The LLM Vote agent can be executed using the script `cot.py`.

Run:

```bash
python example/cot.py
```

This script will launch the Chain-of-Thought based LLM voting agent.

---

## 3. Run the Statistics Vote Agent
To run the statistical voting agent, execute:

```bash
python websocietysimulator/agent/Agent_StatVote.py
```

This will start the statistics-based voting simulation.
